# 🎭 Real-Time Facial Emotion Recognition System

## 📌 Project Overview
This project is a real-time AI system that detects human facial emotions using a webcam feed. It uses DeepFace (a deep learning-based facial analysis framework) combined with OpenCV to process live video and classify emotions instantly.

The system identifies emotions such as:
**Happy, Sad, Angry, Neutral, Fear, Surprise, and Disgust**

---

## 🧠 Key Highlights (Resume Impact)
- Real-time computer vision system using webcam input
- Deep learning-based facial emotion classification
- Automated face analysis without manual feature engineering
- End-to-end Python implementation
- Lightweight and runs locally on CPU

---

## ⚙️ Tech Stack
- Python
- OpenCV
- DeepFace
- TensorFlow
- Keras

---

## 📊 System Workflow
1. Capture video stream from webcam
2. Detect face using DeepFace backend
3. Analyze facial expression using pretrained model
4. Output dominant emotion in real time on screen
